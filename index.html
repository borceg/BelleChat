BelleChat by marknotgeorge

Description

BelleChat is an IRC (Internet Relay Chat) program for Symbian Anna and Nokia Belle smartphones such as the Nokia N8, Nokia E6, Nokia E7, Nokia 500 and Nokia 808 PureView. It has the following features:

  • Written in QT C++ and Qt Quick using the Qt Quick Components for Symbian, thus conforming to the Nokia Belle style quidelines.
  • Allows multiple channel joins.
  • Correctly renders mIRC colours, both for incoming and outgoing messages. mIRC formatting for outgoing messages can be switched off.

BelleChat is written using the Communi library, written by J-P Nurmi, et al. Communi is licenced under the FSF LGPL licence.
